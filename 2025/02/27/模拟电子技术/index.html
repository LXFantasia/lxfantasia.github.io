半导体与PN结

1. 本征半导体

  • 相邻原子之间存在稳定的共价键，导电能力并不强；形成共价键的电子可能发生热激发生成自由电子和空穴，自由电子和空穴一样多，在一定温度下达到动态平衡，称为本征激发。
  • 本征半导体载流子浓度 从该公式可以看出，本征半导体载流子浓度和温度密切相关。因此，对温度稳定性要求高的半导体元件一般不使用本征半导体

2. 杂质半导体

  1. 概念：在本征半导体内掺入少量的杂质元素形成的半导体
  2. N型半导体
    1. 概念：在本征半导体内掺入少量P元素形成的半导体
    2. 载流子：自由电子与空穴，其中自由电子数目远多于空穴，称为多子，空穴称为少子
      思考：半导体中多子受温度影响大吗？少子呢？
      回答：由于温度主要影响的是本征激发速率，而在杂质半导体中本征激发出来的载流子仅占多子很少的一部分，因此半导体中多子浓度受温度影响不大。同样的道理，少子浓度受温度影响很大。因此，主要依靠少子性质的半导体元件受温度影响很大。
      注意：虽然N型半导体中多子为带负电的自由电子，但是N型半导体本身呈电中性，载流子中自由电子远多于空穴是因为正离子不属于载流子的范畴内。
  3. P型半导体（相关内容参考N型半导体条目）
  4. 施主原子与受主原子：五价元素可以提供电子，称为施主原子；三价元素可以提供空穴吸收电子，称为受主原子。

3. PN结

  1. 形成：将P型半导体和N型半导体放在一起，随着载流子的扩散，会逐渐形成一个由N指向P的电场，阻止载流子进一步扩散
  2. PN结（耗尽层、阻挡层、空间电荷区）：PN交界处载流子浓度很低的一块区域
    注意：阻挡层形成后，并不意味着扩散运动不会发生，如果载流子能够获得足够多的能量跨越势垒，则扩散运动仍然可以发生
  3. 扩散运动与漂移运动：扩散运动是载流子因浓度不同引起的运动，漂移运动是载流子在电场力的作用下进行的运动；载流子扩散运动形成的电流称为扩散电流，漂移运动形成的电流称为漂移电流
  4. 对称结与非对称结：与两边掺杂浓度有关

4. PN结的性质

  1. 单向导电性
    1. 外加正向电压：外电场与内电场反向，使空间电荷区变窄，从而削弱了内电场，有助于扩散运动的恢复，同时削弱了少子的漂移运动，从而形成正向电流，使PN结导通（注意加限流电阻限制最大电流）
    2. 外加反向电压：外电场与内电场同向，使空间电荷区变宽，加强了内电场，进一步阻止了扩散运动（同时加强了漂移运动，会因此形成一个反向饱和电流，对温度非常敏感）
  2. PN结的电流方程：
    $$i = I_s(e^{\frac{U}{U_T}} - 1)$$
    其中$I_S$是反向饱和电流，$U_T$是温度当量，满足$U_T = \frac{kT}{q}$，其中q表示电子电量，k表示玻尔兹曼常量，T是热力学温度，室温下（T = 300K）$U_T = 26mV$
  3. PN结的伏安特性
    1. 正向特性
    2. 反向特性
    3. 反向击穿
      1. 雪崩击穿：杂质离子掺杂浓度较低时，PN结耗尽层足够宽的情况下漂移运动的载流子速度变得十分快，冲击共价键导致形成共价键的电子被释放出来形成自由电子，从而导致雪崩式的反应。温度越高，雪崩击穿所需的击穿电压越高（温度升高，晶格振动幅度增大，干扰自由电子的加速）
      2. 齐纳击穿：杂质离子掺杂浓度很高时，PN结耗尽层足够窄，共价键里的电子直接被高场强拉出来。温度越高，齐纳击穿所需的击穿电压越低
        注意： 反向击穿不一定导致PN结熔毁，PN结烧坏的原理是反向击穿功率过大导致发热严重（在烧坏之前反向击穿是可逆的），称为二次击穿导致不可逆
  4. PN结的电容效应
    1. 势垒电容：PN结的宽度与外加反向电压之间存在关系
    2. 扩散电容：外加正向电压时，非平衡少子电量和正向电压之间存在关系

2. 放大原理

  1. 发射结的情况：发射区自由电子向基区扩散，基区空穴向发射区扩散，这两部分共同构成了$I_E$，由于发射区载流子浓度远高于基区，所以$I_{EN}$占主导地位
  2. 基区的情况：从发射区扩散过来的大量自由电子比基区的多子还多，必然会从基区向集电区扩散，同时与基区的空穴结合，并在基区有少量产生，与此同时，集电结反偏，会把基区飞过来的自由电子向集电极运输
  3. 集电区的收集：

    $$
    I_E = I_{EN} + I_{EP} \
    I_C = I_{CN} + I_{CBO} \
    I_B = I_{BN} + I {EP} - I{CBO}
    $$
  4. 放大系数
    1. 共射直流放大系数：$\overset{-}{\beta} = \dfrac{I_{CN}}{I_{BN}} \approx \dfrac{I_C}{I_B}$
    2. 共射交流放大系数：$\beta = \frac{\Delta i_C}{\Delta i_B} \approx \overset{-}{\beta}$
    3. $I_C = \overset{-}{\alpha} I_E + I_{CBO}$
    4. $I_{CEO} = 1 + \beta I_{CBO}$
    5. $I_{CBO}$
    6. 共基直流放大系数：$\overset{-}{\alpha}=\dfrac{I_C}{I_E} = \dfrac{\overset{-}{\beta} }{1+\overset{-}{\beta}}$
    7. 共基交流放大系数（定义类比共射）
    8. 共集直流放大系数：$1 + \overset{-}{\beta}$
    9. 共集交流放大系数（定义类比共射）

4. 共射特性曲线

  1. 输入特性（$I_B - U_{BE}$）（前提：$U_{CE}$为常数）：就应该长PN结那样
  2. 输出特性（$I_{C} - U_{CE}$）（前提：$I_B$为常数）（关键：集电结是否反偏）
    1. 放大区：发射结正偏，集电结反偏，$I_C与I_B成比例$
    2. 截止区：发射结反偏，集电结反偏，$I_C = I_{CEO} \approx 0$
    3. 饱和区：发射结正偏，集电结正偏，$\beta I_B > I_{Cmax}, I_C = I_{Cmax}$

4.5 放大电路的失真分析

  1. 截止失真：（对NPN管而言）原因是Q点设置得太低，导致电压真实值$u_{BE}$小于管子的截止电压，从而导致$i_B、i_C$出现底部失真，由于$u_{CE}= V_{CC} - i_CR_C$，故$u_{CE}$出现的是顶部失真。对PNP管正好反过来。
    解决方案：只能调整输入回路。增大$V_{BB}$或减小$R_B$使得Q点抬高。
  2. 饱和失真：（对NPN管而言）原因是Q点设置得太高，导致放大后电流达到输出特性曲线的饱和区。从而导致$i_C$出现顶部失真，$U_{CE}$出现底部失真，PNP管正好反过来。
    解决方案：对于输入回路而言，可以减小$V_{BB}$或增大$R_B$，或者换用β更小的晶体管；对于输出回路而言，可以减小$R_C$。
  3. 非线性失真：信号太大导致线性等效出现较大误差。第五章介绍的反馈可以减小非线性失真。
  4. 如果同时出现截止失真和饱和失真，说明$V_{CC}$设计得不合理，导致信号可调范围太小。应该适当增大$V_{CC}$
  5. 交越失真（针对乙类放大电路而言），本质是截止失真。

1. 放大电路的构成

  1. 放大的特征：放大的是（交流）功率
  2. 放大的本质：能量的控制和转换
  3. 放大的必要条件：存在有源的控制元件
  4. 放大的前提条件：不失真
  5. 放大的测试：正弦波

2. 怎样构建放大电路

  1. 目标：把小功率信号变成大功率信号
  2. 条件：1、控制元件；2、电源
  3. 技术路线：
    1. 元件工作在放大状态下
    2. （以三极管为例）小信号$\to u_{BE} \to i_B$
    3. 存在一个合理的输出过程
      注意： 为了使小信号能控制输出信号，要先加一个直流电源使三极管能够导通，同时要加一个电阻限制发射回路的最大电流；然后，应使三极管工作在放大状态下，即使集电结反偏；然后，为了得到放大后的信号，还需要把交流的电流信号用合适的方法（比如元件）转化为我们需要的信号
      注意： 是否设置了合适的静态工作点；

      直接耦合的问题：1、电阻分压太大；2、需要直流与交流共同作用

4. 放大电路的分析方法

  1. 直流通路：将交流电源置零得到的电路（电容断路、电感短路）作用：直接分析放大电路的静态性质
  2. 交流通路：交流信号流经的通路（直流源置零（把Vcc变成接地），电容近似于短路，能合并的接地线要画到一起）
  3. 图解法：
    1. 找静态工作点：分析直流通路，找特性曲线交点
    2. 增加交流信号
    3. 在输出特性曲线族上找到变化的iB对应的曲线
    4. 在输出特性图上找负载曲线与输出特性曲线的交点
  4. 放大电路的线性等效：$\delta i_C, \delta i_B$具有近似的线性关系，可以进行近似的线性等效（使用等效电阻和受控源进行近似的线性等效）
  5. 等效电路法
    1. 分析直流通路
      1. 找Q点（注意分析Q点的合理性（不能工作在饱和区））
      2. 推出$r_{be} = r_{bb’} + (1+\beta)\frac{U_T}{I_{EQ}}$（$r_{bb’}$是发射结的体电阻）
    2. 分析交流通路
      1. 画交流通路，找微变等效或者h参数等效
  6. h参数等效：
    1. h参数
    2. 简化的h参数（微变等效）（be之间加$r_{be}$（动态等效电阻），ce之间加流控电流源）
  7. 共射放大电路动态参数的分析
    1. 电压放大倍数$A_u = -\dfrac{\beta R_e}{R_b + r_{be}} = -\dfrac{\beta R_o}{R_i}$
    2. 电流放大倍数$A_i = \beta$
    3. 输入电阻$R_i = R_b + r_{be}$
    4. 输出电阻$R_o = R_e$
  8. 基本共射放大电路的失真
    1. 截止失真（Q点太小，导致电压出现小于发射极开启电压的情况）
    2. 饱和失真（Q点太大，导致Ic超过了Icmax）

5. 放大电路Q点的稳定

  1. 影响Q点的因素：温度、电源的波动、元器件的老化
  2. 稳定Q点的思路：最主要解决的是温度的影响
    解决方案：用输出影响输入，形成负反馈状态，具体操作是在e和接地点之间加入电阻，这样一来$I_C$增大，$I_E$也增大，引起电阻分压增大，导致$u_BE$减小，从而减小$I_C$的增加
  3. Q点稳定的电路
    1. 构成
    2. 分析
      e.g. 分压式偏置（图c） $T\uparrow \to I_B \uparrow \to I_C \to I_{E}\uparrow \to U_E \uparrow \to U_{BE} \downarrow \to I_B \downarrow$实现负反馈
    3. 加入$R_e$的共射放大电路的动态参数分析
    4. 旁路电容：交流状态下将$R_E$短路，避免$R_E$对$A_{uu}$的影响（不用电容的话，可以使用差分放大电路）

多级放大电路

1. 耦合方式

  1. 直接耦合 第一级与第二级之间没有其他的东西，直接连起来了（图一实际上不合理，T1的集电极和T2的基极并联，使得T1无法满足uce与ube的关系。图2~4是对图1问题的改良）
    优点：低频特性好，便于集成；
    缺点：调试困难，易受干扰
  2. 阻容耦合
    优点：便于调试，各级独立，稳定Q点容易
    缺点：低频特性差，难集成，所以常见于分立电路
  3. 变压器耦合 两级之间通过变压器耦合
    优点：直流状态互不干扰，可以进行阻抗变换
  4. 光电耦合 优点：抗干扰，绝缘性好

差分放大电路

1. 直接耦合放大电路存在的问题

  1. 零点漂移
    1. 产生原因：环境温度无法保持恒定（还会受元器件和电源的影响）
    2. 温漂：零点受温度影响发生漂移的现象
    3. 解决方案：添加负反馈 缺点：为稳定零点需要增大Re（为什么，自己推），然而Re越大UE越大，进而会导致UBE下降，导致iB下降，导致iC下降，这是我们不希望看到的
    4. 进一步的解决方案：期望引入一个可变电源，使得其可以随着直流源的波动而波动，从而抵消零点漂移。为达到这一目的，我们可以在相应的位置添加一个镜像电路 然后将途中的两个RE公用为一个 再将途中的VBB共用为一个

  1. 电路结构：
    共模信号与差模信号：前者是大小与方向相同；后者是大小相同，方向相反。差分放大电路可以放大差模信号，同时抑制共模信号（干扰信号一般是共模信号）

放大电路的频率响应

1. 频率响应的基本概念

  1. 频率对放大电路输出产生影响的原因:低频主要是电路中电容产生的影响（所以直接耦合电路不存在低频的衰减），高频主要是PN结的节间电容产生的影响
  2. 下限频率、上限频率与通频带
  3. 幅频特性与相频特性：前者是$A_u$与f的关系，后者是$\varphi$与f的关系
  4. 高通电路
    1. 传输特性： $A_u = \dfrac{R}{R + \dfrac{1}{j\omega C}} = \dfrac{j\dfrac{f}{f_L}}{1+j\dfrac{f}{f_L}}$（后者称为标准形式）
    2. 下限频率：$f_L = \dfrac{1}{2\pi RC} \Leftrightarrow \omega = \dfrac{1}{RC}$（信号频率大于下线频率幅值就处于通频带内，可以简单理解为频率越高越通）
  5. 低通电路：
    1. 传输特性：$A_u = \dfrac{\dfrac{1}{j\omega C}}{R + \dfrac{1}{j\omega C}} = \dfrac{1}{1+j\dfrac{f}{f_H}}$
    2. 上限频率：$f_H = \dfrac{1}{2\pi RC} \Leftrightarrow \omega = \dfrac{1}{RC}$
      注意： 高通对应$f_L$，低通对应$f_H$

2. 波特图

  1. 功能：反映电路的幅频特性与相频特性
  2. 坐标参数：
    1. 幅频特性曲线：横轴为$lgf$，纵轴为$20lg|A_u|$，单位为分贝（dB）。纵坐标为正表示增益，纵坐标为负表示衰减
    2. 相频特性曲线：横轴为$lgf$，纵轴为$\varphi$

放大电路中的反馈

1. 反馈的基本概念

  1. 反馈的概念：输出量或其部分通过一定的形式作用到输入回路，用来影响输入量的措施
  2. 净输入量：输入量与反馈量的共同作用
  3. 方框图
  • 任何负反馈放大电路均可以使用该图描述，满足$X_i’ = X_i - X_f$。
  • 基本放大电路的放大倍数$A = \dfrac{X_o}{X_i’}$；反馈系数$F = \dfrac{X_f}{X_o}$；
  • 反馈电路的放大倍数（闭环放大倍数）$A_f = \dfrac{X_o}{X_i} = \dfrac{X_o}{X_i’ + X_f} = \dfrac{A}{1 + AF}$
  • 环路放大倍数：$AF = \dfrac{X_f}{X_i’}$
  • 反馈深度：$|1 + AF|$，当该值较大时，称为深度负反馈
  • 深度负反馈下的$A_f = \dfrac{A}{1 + AF} \approx \dfrac{1}{F}$

  2. 直流反馈与交流反馈

2. 反馈的判断

  1. 是否存在反馈：观察放大电路输出回路与输入回路是否有通路相连接
  2. 反馈存在于直流还是交流：观察直流通路和交流通路中是否存在反馈回路

4. 深度负反馈放大电路

  1. 深度负反馈放大电路概念：在负反馈放大电路的一般表达式中，若$|1 + AF| >> 1$，则$A_f = \dfrac{A}{1 + AF} \approx \dfrac{1}{F} = \dfrac{X_o}{X_f}$，再结合$A_f = \dfrac{X_o}{X_i}$，说明$X_i \approx X_f$。可见，深度负反馈的实质是在近似分析中忽略净输入量。

5. 负反馈放大电路的稳定性

  1. 自激振荡产生的原因：高频下的晶体管与低频下的耦合电容会使A和F发生相移，使反馈信号发生相移，在某种频率下A与F的相移之和为180°，使得负反馈变成正反馈
  2. 自激振荡的平衡条件$|AF| = 1$（该条件与相位和为180°要同时满足），起振条件$|AF| > 1$
  3. 放大电路是否能发生自激振荡的判断方法：
    1. 看是否有频率满足A和F的相移之和为180°
    2. 若有，看此时|AF|是否大于1
    3. 多级放大电路从三级开始起振（因为每级放大电路的最大相移是90°）
  4. 消除自激振荡的方法：
    1. 滞后补偿：让20lgA迅速下降，在相移达到180之前让20lgA降到0以下
